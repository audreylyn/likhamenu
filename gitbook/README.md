# WebGen AI Documentation

Welcome to the WebGen AI documentation. This guide will help you set up and use the WebGen AI system to create and manage websites with AI-assisted content generation.

## What is WebGen AI?

WebGen AI is a powerful React-based website generator that allows you to:

- Create multiple types of websites (Portfolio, E-commerce, Landing Page, Restaurant, Service Agency, Event/Conference, Blog, and Custom)
- Generate content automatically using Google Gemini AI
- Manage multiple websites from a single admin dashboard
- Integrate contact forms, order tracking, and chat support
- Customize visual styles, themes, and content sections

## Documentation Structure

This documentation is organized into the following sections:

1. **Installation Guide** - Step-by-step instructions for setting up WebGen AI on a new computer
2. **Usage Guide** - Step-by-step instructions on how to use the system after installation

## Prerequisites

Before you begin, ensure you have:

- Node.js (v18 or higher) and npm installed
- A Supabase account
- A Google Gemini API key
- A Google account (for optional Apps Script integrations)
